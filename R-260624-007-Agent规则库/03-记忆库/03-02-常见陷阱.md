# 03-02 常见陷阱（元规则/SOP未覆盖项）

> 元规则和SOP中已记录的大量陷阱见：01-02 Self-Correction Log、01-04 不编造证据铁律、02-03 检查清单。

## python-docx保存前确保目录存在

用python-docx保存文件到新路径时，目标目录必须预先存在（`mkdir -p`）。
FileNotFoundError是最常见的低级错误，放在内容写入循环之外处理。

## Word生成§9.1样式清理必须在内容写入之前

```python
# §9.1 样式清理（强制执行）
for i in range(1, 10):
    for suffix in ['', ' Char']:
        style_name = f'Heading {i}{suffix}'
        if style_name in doc.styles:
            s = doc.styles[style_name]
            s.font.color.theme_color = None
            s.font.color.rgb = RGBColor(0, 0, 0)
```

这段代码必须在任何`doc.add_paragraph()`或`doc.add_heading()`之前执行。否则后续写入的内容会继承Word默认的蓝色Heading样式。
