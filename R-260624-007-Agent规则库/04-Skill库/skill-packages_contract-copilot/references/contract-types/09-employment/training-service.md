# 培训服务期协议

## 合同概述

培训服务期协议用于约定用人单位提供专项培训后，劳动者在约定期限内继续履职以及违约责任的承担规则。

适用边界：

- 必须存在可证明的专项培训投入。
- 服务期应与培训投入、岗位需求匹配。
- 违约金应与未履行期间和未摊销投入挂钩。
