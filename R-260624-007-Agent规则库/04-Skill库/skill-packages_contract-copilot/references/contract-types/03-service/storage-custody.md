# 保管仓储合同

## 合同概述

保管合同是保管人保管寄存人交付的保管物并返还原物的合同。仓储合同属于有偿保管的特殊类型，通常由仓储经营人提供储存服务。

**审查重点差异**：

- 保管合同可有偿也可无偿；仓储合同通常有偿。
- 仓储合同对仓单、出入库、保管条件、风险转移要求更高。
- 危险品、冷链品、贵重物品的审查标准应单独提升。

- 行业监管规则（危化品、冷链、消防等，按场景适用）
